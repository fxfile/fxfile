2. 소개

  여러분들 개개인의 소중한 파일 및 폴더들을 편리하게 탐색하고 관리할 수 있는 유용한 전문 파일 관리 프로그램입니다. 파일 실행, 폴더 탐색, 복사, 잘라내기, 붙여넣기, 드러그 앤 드롭, 등의 기본 기능에 강한 flyExplorer입니다. 윈도우즈 탐색기(이하 탐색기)의 기본 기능들은 그대로 유지한 상태에서 사용자들로부터 요구된 기능들이나 필요한 기능들을 첨가하여 확장시켜 나가는 프로그램입니다. 유사한 프로그램이 여러 존재하지만, 타 프로그램과의 차별성은 윈도우즈 탐색기의 사용성을 그대로 둔 상태이기 때문에 처음 사용자가 접근하기 아주 쉽다는 것과 이와 더불어 많은 확장 기능들을 제공한다는 것이다. 또한, 폴더 및 파일의 변화를 감지하는 자동 갱신 기능도 강력하게 동작한다. 
  flyExplorer를 사용하시는 데에 있어서 문의할 사항이 있으면 홈페이지의 질문과 답변 게시판에 글을 남겨 주시고, 필요한 기능이 있어 기능을 제안하고 싶다면 홈페이지의 기능 제안 게시판에 글을 남겨 주시기 바랍니다. 또한 사용하시다가 버그가 발견되었다면 홈페이지의 버그 레포트 게시판에 글을 남겨 주시면 다음 업데이트 버전에서 패치될 것입니다.
  다른 어느 파일 관리 프로그램보다 더 뛰어나고 더 강력하고 더 빠른 기능을 제공하여 세계 최강의 파일 관리 프로그램을 개발하도록 하겠습니다. 또한 국내외에 널리 사용될 수 있도록 피땀흘려 열심히 노력하겠습니다. 여러분들의 많은 성원 부탁드리겠습니다.

  차별된 기능으로써 승부하겠습니다. 편리하게 파일과 폴더를 관리하고 탐색하십시오!!!

3. 사용 환경

  1) 하드 웨어(Hardware)
     중앙처리장치(CPU) : Pentium II 이상 권장
     메모리(Memory) : 64MB 이상 권장
     그래픽 카드(Grahpic Card) : 800 × 600 해상도, 16 bit 컬러 이상
     하드 디스크(Hard disk) : 여유 공간 10MB 이상
 
  2) 소프트웨어(Software)
     운영체제(Operating System)
     Ansi 버전 : 9x/NT 계열 모두 지원(Windows 98, NT, ME, 2000, XP, 2003, Vista, 2008, 7, 8, 2012 이상)
     Unicode 버전 : NT 계열 지원(Windows NT, 2000, XP, 2003, Vista, 2008, 7, 8, 2012 이상)
     인터넷 익스플로러(Internet Explorer) : 5.0 이상
