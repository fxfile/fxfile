■ 사용자 인터페이스

  flyExplorer의 사용자 인터페이스는 기본적인 사용자 인터페이스는 윈도우즈 탐색기와 흡사하여 처음 접하는 사용자들도 인터페이스들이 쉽게 눈에 들어올 것입니다. 각 인터페이스는 다음 그리과 같이 구성되어 있습니다.

 

1. 주요 사용자 인터페이스

  주요적인 사용자 인터페이스는 위 그림과 같이 구성되었고, 그림의 번호별로 각 명칭은 다음과 같습니다.

2. 각 인터페이스 설명

  1) 메인 메뉴 : 특정 기능을 수행하기 위해 명령을 내릴 때 사용되는 인터페이스입니다.

  2) 메인 도구 : 메인 메뉴의 기능 중 가장 많이 사용되는 기능을 더 빠르게 수행하기 위해 따로 모아둔 것으로써, 버튼으로 명령 수행이 가능합니다.

  3) 연결 항목 도구 : 연결 항목 도구에서 폴더의 이동 및 파일을 수행할 수 있는 기능들로 구성되어 있습니다.

  4) 주소 표시줄 : 주요 폴더를 쉽게 이동하고 경로명을 입력하여 해당 폴더로 이동하거나 해당 파일을 실행하는 인터페이스이다. 또한 자동 완성 기능이 있어 더욱 편리하게 쓰일 수 있습니다.

  5) 경로바 : 상위 폴더를 단계적으로 이동하기 쉽게 구성되어 있습니다.

  6) 드라이브 도구 : 드라이브간 이동을 빠르게 하기 위한 기능을 담당하고 있습니다.

 7) 활성바: 분할된 창에서 활성화된 창을 표시해 주는 기능을 담당하고 있습니다.

  8) 폴더창 : 폴더로만 구성되어 있어 폴더만 탐색이 가능하고, 탐색창과 더불어 가장 많이 사용되는 사용자 인터페이스이다. 기존 탐색기는 폴더창이 탐색창 왼쪽에만 위치하였지만, flyExplorer는 사용자 편의를 따라 어느 위치든 도킹할 수 있고 띄울 수도 있다.

  9) 컨텐츠 : 탐색창의 선택한 파일, 폴더 정보를 표시해주며 연결 항목도 표시해 주는 인터페이스이다.

  10) 탐색창 : 파일, 폴더를 탐색하는 주된 인터페이스이며, 거의 모든 작업이 폴더창과 탐색창이 많이 이루어진다.

  11) 상태 표시줄 : 현재 탐색하고 있는 폴더의 상태를 표시하는 인터페이스이다. 중간에서 약간 오른쪽에 보면 현재 탐색하고 있는 드라이브가 나타내며, 드라이브의 사용 가능한 용량도 쉽게 파악할 수 있다.

 

3. 이외 인터페이스 설명

  1) 팝업 메뉴 : 메인 메뉴에 있는 명령을 더욱 빠르게 수행하거나 특정 상황에서 명령을 수행할 경우, 마우스 오른쪽 버튼을 눌러 명령을 수행하는 인터페이스이다.

  2) 검색창 : 특정 폴더내에 파일, 폴더를 검색하기 위한 창으로써, 문자와 와이드 카드(*, ?)를 혼합하여 검색할 수 있습니다.

  3) 작업창 : 검색창에서 검색된 결과를 보여주는 인터페이스이다. 추후에 ftp의 기능이 추가되면 다운로드와 업로드의 큐 상태를 보여주는 기능도 담당할 것이다.
