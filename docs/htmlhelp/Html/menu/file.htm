  + 일괄적 새로 만들기

이름 바꾸기

일괄적 이름 바꾸기

선택한 항목의 이름을 일괄적으로 바꿉니다.

   + 일괄적 이름 바꾸기

선택한 항목의 이름을 일괄적으로 모두 바꿉니다.

   + 파일만 일괄적 이름 바꾸기

선택한 항목 중 파일만 일괄적으로 이름을 바꿉니다.

   + 폴더만 일괄적 이름 바꾸기

선택한 항목 중 폴더만 일괄적으로 이름을 바꿉니다.

이미지 포멧 바꾸기

선택한 이미지 파일의 포멧을 바꿉니다.

속성/날짜 바꾸기

선택한 파일이나 폴더의 속성 및 날짜를 바꿔줍니다.

등록 정보

선택한 항목의 등록 정보를 자세히 보여줍니다.

파라미터 실행

선택한 항목이 실행 파일이라면 파라미터 입력과 함께 실행합니다.

CRC Checksum 생성

파일 분할 한 파일을 여러 조각의 파일로 분할합니다.
파일 결합 조각난 파일을 다시 한 파일로 결합 합니다.
파일 합치기 여러 파일을 하나의 파일로 합칩니다.

내보내기

  + 파일 목록 만들기

  + 텍스트 문서 합치기

  + 파일 첨부하여 메일 보내기
