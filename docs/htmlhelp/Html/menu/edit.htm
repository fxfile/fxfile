잘라내기

선택한 항목을 클립보드에 잘라냅니다.

복사

선택한 항목을 클립보드에 복사합니다.

이름 복사

선택한 항목의 이름을 클립보드에 복사합니다.

파일명 복사

개발자 경로명 복사

URL 경로명 복사

선택한 항목의 URL 경로명을 클립보드에 복사합니다.

붙여넣기

클립보드로 잘라내거나 복사된 파일 및 폴더를 현 폴더에 이동시키거나 복사합니다.

사본 붙여넣기

클립보드로 잘라내거나 복사된 파일 및 폴더를 현 폴더에 사본으로 이동시키거나 복사합니다.

바로 가기 붙여넣기

클립보드로 잘라내거나 복사된 파일 및 폴더를 현 폴더에 바로가기를 만듭니다.

선택하여 붙여넣기 클립보드 내용을 선택하여 붙여 넣습니다.

수집함 작업

  + 현 위치로 복사

  + 현 위치로 이동

수집함에 수집된 파일 및 폴더를 현 위치로 이동합니다.

  + 삭제

수집함에 수집된 팡리 및 폴더를 삭제합니다.

  + 클립보드로 잘라내기

수집함에 수집된 파일 및 폴더를 클립보드로 잘라냅니다.

  + 클립보드로 복사

수집함에 수집된 팡리 및 폴더를 클립보드로 복사합니다.

  + 수집함 관리

수집된 파일 및 폴더의 목록을 파일 수집함 대화 상자로 보여줍니다.

  + 수집함 목록 제거

수집된 파일 및 폴더의 목록을 제거합니다.

수집 그룹 작업

  + 현재 폴더 수집 그룹 추가

  + 수집 그룹 관리

수집 그룹을 관리합니다.
