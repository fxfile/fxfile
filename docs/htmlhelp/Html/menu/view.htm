메인 도구를 보여주거나 숨깁니다.

  + 드라이브 단추

  + 연결 항목 단추

  + 주소 표시줄

  + 경로바

  + 활성바

활성바를 보여주거나 숨깁니다.

  + 주소 표시줄 펼치기

  + 드라이브 이름 짧게 표시

  + 드라이브 이름 길게 표시

  + 드라이브 여러줄로 표시

드라이브 도구의 드라이브 표시가 한 줄이 넘어갈 경우 여러줄로 표시합니다.

  + 분할창에 각각 할당

드라이브 도구를 분할창에 각각에 표시합니다.

  + 경로바의 왼쪽에 위치

드라이브 도구를 분할창에 각각 표시할 경우 경로바의 왼쪽에 위치합니다.

  + 경로바의 오른쪽에 위치

드라이브 도구를 분할창에 각각 표시할 경우 경로바의 오른쪽에 위치합니다.

  + 드라이브 새로 고침

드라이브 도구의 드라이브를 새로 고칩니다.

  + 연결 항목의 이름을 표시하지 않음

연결 항목 도구의 이름을 표시하지 않습니다.

  + 연결 항목의 이름을 표시

연결 항목 도구의 이름을 표시합니다.

  + 연경 항목 여러줄로 표시

연결 항목 도구의 연결 항목 표시가 한 줄이 넘어갈 경우 여러줄로 표시합니다.

  + 연경 항목 새로 고침

연결 항목 도구의 연결 항목을 새로 고칩니다.

  + 도구 이동 잠금

  + 사용자 지정

창 모음

여러 창을 보여주거나 숨김니다.

  + 폴더창

폴더를 탐색하기 위한 폴더창을 보여주거나 숨깁니다.

  + 검색창

파일 및 폴더를 검색하기 위한 검색창을 보여주거나 숨깁니다.

  + 작업창

검색창에서 검색된 결과를 보기 위한 작업창을 보여주거나 숨깁니다.

그림 보기

많은 종류의 그림을 독립된 창이나 윈도우 탐색기처럼 컨텐츠에 보여줍니다.

  + 확대

그림을 확대 시킵니다.

  + 축소

그림을 축소 시킵니다.

  + 100% 크기

그림 크기의 100%로 조정합니다.

  + 비율적으로 크기 조정

그림 크기 비율적으로 창을 조정합니다.

  + 탐색창에 도킹하기

탐색창의 컨텐츠 부분에 도킹시킵니다. 단, 단일창일 경우에만 도킹이 가능합니다.

  + 자동 숨김

탐색창에서 선택한 파일이 그림 파일이 아닐 경우 자동으로 숨깁니다.

  + 크기 고정

현재 그림 크기를 고정시킵니다.

  + 창 크기로 고정

그림을 창 크기에 맞춥니다.

  + 100% 크기로 고정

그림을 무조건 100% 크기로 보여줍니다.

  + 썸네일과 같게 고정

썸네일과 같은 크기로 보여줍니다.

  + 타이틀바에 파일명만 보이기

그림 보기 창의 타이틀바에 파일명만 표시합니다.

  + 바탕 화면의 배경 설정

바탕 화면의 배경 그림으로 설정합니다.

  + 보이기/숨기기

그림 보기 창을 보이기/숨기기 기능을 토글시킵니다.

  + 숨기기

그림 보기 창을 숨깁니다.

큰 아이콘

작은 아이콘

간단히

탐색창의 항목을 간단히 표시합니다.

자세히

탐색창의 항목을 이름, 크기, 종류, 수정한 날짜 등으로 자세히 표시합니다.

썸네일 이미지

탐색창의 항목을 썸네일 이미지로 표시하여 이미지 파일을 작게 볼 수 있습니다.

아이콘 정렬

아이콘 정렬에 관련된 명령의 모음입니다.

  + 이름순

탐색창의 항목을 이름순으로 정렬합니다.

  + 확장자순

탐색창의 항목을 확장자순으로 정렬합니다.

  + 종류별

탐색창의 항목을 종류별로 정렬합니다.

  + 크기순

탐색창의 항목을 크기순으로 정렬합니다.

  + 날짜순

탐색창의 항목을 날짜순으로 정렬합니다.

  + 속성순

탐색창의 항목을 속성순으로 정렬합니다.

  + 오름 차순

탐색창의 항목을 오른 차순으로 정렬합니다.

  + 내림 차순

탐색창의 항목을 내림 차순으로 정렬합니다.

열 선택

탐색창에서 자세히 스타일 경우 해당 열을 보이거나 숨깁니다.

  + 이름

탐색창에서 이름 열을 보이거나 숨깁니다.
  + 확장자

탐색창에서 확장자 열을 보이거나 숨깁니다.

  + 크기

탐색창에서 크기 열을 보이거나 숨깁니다.

  + 종류

탐색창에서 종류 열을 보이거나 숨깁니다.

  + 수정한 날짜

탐색창에서 수정한 날짜 열을 보이거나 숨깁니다.

  + 속성

탐색창에서 속성 열을 보이거나 숨깁니다.
  + 세부 설정

탐색창의 열을 세부적으로 설정합니다. 윈도우 탐색기에서 제공하는 열도 추가 가능합니다.

새로 고침

항목들을 새로 고칩니다.
