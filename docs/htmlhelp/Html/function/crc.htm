■ CRC Checksum

  CRC Checksum 기능은 파일의 유효성을 검사하는 기능입니다.

 

1. CRC Checksum 파일 생성

  탐색창에서 체크섬 파일을 생성할 파일을 선택한후, "파일" 메뉴의 "CRC Checksum 생성"을 선택하면 CRC Checksum 대화 상자가 띄워집니다. CRC 방식과 각 파일별 CRC 파일 생성 옵션을 설정합니다. CRC 방식은 SFV, MD5 두 가지 방법을 제공합니다. 각 파일별 CRC 파일 생성은 탐색창에서 선택한 파일별로 CRC 파일을 생성할 것인지 아니면 하나의 CRC 파일로 만들 것인지 여부입니다. 이 후 생성 버튼을 누르면 CRC 파일이 생성됩니다.

 

2. CRC Checksum 검사

  탐색창에서 SFV이나 MD5 파일을 선택한 후, "파일" 메뉴의 "CRC Checksum 검사"를 선택하면 CRC Checksum 검사 대화 상자가 띄워집니다. 검사 버튼을 누르면 검사가 진행됩니다. 모두 완료되어 결과에 보면 CRC 파일 안에 있는 코드와 직접 검사한 코드가 일치하는 볼 수 있습니다.
