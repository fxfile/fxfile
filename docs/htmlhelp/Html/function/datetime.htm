■ 일괄적 날짜/속성 바꾸기

  파일이나 폴더의 날짜 및 속성을 일괄적으로 바꾸주는 기능입니다.

 

1. 사용 방법

  탐색창에서 바꾸고자 하는 파일이나 폴더를 선택한 후,  메인 메뉴 "파일"의 "속성/날짜 바꾸기"를 선택하여 "일괄적 속성/날짜 바꾸기" 대화 상자를 띄워 해당 옵션들을 설정하고 "확인" 버튼을 선택합니다. 바꾸고자 하는 대상이 폴더일 경우에는 하위 폴더의 단계를 설정할 수 있는 옵션도 제공합니다.

 

2. 옵션

    1) 속성 : 파일에는 기록, 숨김, 읽기 전용, 시스템 등 약 4가지의 속성이 있으며, 바꾸고자하는 해당 옵션을 설정합니다.

    2) 날짜 : 파일에는 생성, 마지막 수정, 최근 접근 날짜가 있으며, 해당 날짜나 시간을 설정합니다.

    3) 하위 폴더 : 이 옵션을 설정하면 바꾸고자하는 대상이 폴더일 경우 그 하위에 있는 파일이나 폴더의 속성이나 날짜를 바꿀 수 있습니다. 또한 하위 레벨까지 설정할 수 있어 하위 몇 단계까지 바꿀 수 있는 기능까지 제공합니다.

    4) 적용할 종류 : 파일만 바꿀지, 폴더만 바꿀지, 아니면 모두 다 바꿀 것인지의 옵션을 설정합니다.
