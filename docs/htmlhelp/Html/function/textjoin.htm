■ 텍스트 문서 합치기

  탐색창에서 선택한 텍스트 문서들을 하나의 텍스트 문서로 합쳐주는 기능입니다.

 

1. 사용 방법

  탐색창에서 합칠 텍스트 문서 파일를 선택한 후, 메인 메뉴 "파일"중 "내보내기"의 "텍스트 문서 합치기"를 선택하면, "텍스트 문서 합치기" 대화 상자가 띄워집니다. 또한 단축키 Ctrl+J를 눌러서 띄울 수도 있습니다. "텍스트 문서 합치기" 상자를 보면 "인코딩"이 있는데, 이것은 합쳐질 문서의 인코딩 방식을 정하는 것이다. 합칠 텍스트 문서의 순서를 바꾸고 싶다면 항목을 선택한후, 위아래의 버튼을 누르면 된다. 옵션을 설정한 후, "합치기" 버튼을 누르면 저장할 텍스트 문서 파일명을 지정하는 상자가 먼저 띄워져, 파일명을 입력하면 텍스트 문서가 합쳐진다.

 

2. 옵션 설정

  편의에 따라 옵션을 지정하여 텍스트 문서를 합칠 수 있습니다. 지원하는 옵션은 다음과 같습니다. 단, 윈도우즈 9x 계열에서는 인코딩 방식을 선택할 수 없습니다.

옵션

설명

인코딩

결과 텍스트 문서의 인코딩 방식.
