■ 파일 분할/결합

  파일 분할 기능을 한 파일을 여러 조각으로 나누는 기능이고, 파일 결합을 거꾸로 여러 조각난 파일을 하나의 파일로 결합해 주는 기능입니다. 또한, 파일 합치기 기능을 여러 파일을 하나의 파일로 내용을 합쳐주는 기능입니다.

 

1. 파일 분할

   분할할 파일을 선택한 후 "파일 메뉴 > 파일 분할" 메뉴를 수행하면 된다. 크기 또는 파일 수에 의해 분할할 수도 있다. 크기 분할 옵션은 일정한 파일 크기로 분할하는 옵션이고, 파일 수 분할 옵션은 파일 수를 제한하여 분할하는 옵션이다.

2. 파일 결합

   분할된 파일 중 첫 파일을 선택한 후 "파일 메뉴 > 파일 결합" 메뉴를 수행하면 된다.

3. 파일 합치기

   합치고자 하는 파일을 선택한 후 "파일 메뉴 > 파일 합치기" 메뉴를 수행하면 된다. 단, 합치고자 하는 파일의 순서를 확인한 후 합치기 명령을 수행한다.
