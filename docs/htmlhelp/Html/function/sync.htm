■ 폴더 비교/동기화

  폴더 비교 기능은 분할 된 창끼리 서로 폴더 내용을 비교하여 서로 다른 내용의 파일을 모두 선택해 줍니다. 폴더 동기화 기능은 두 폴더 간의 비교를 통해 설정에 맞추어 동기화해 주는 기능입니다.

 

1. 폴더 비교하기

   수평 분할 또는 수직 분할 된 상태에서 "창 메뉴 > 폴더 비교하기" 메뉴를 수행하면 내용이 다른 파일을 모두 선택해 줍니다.

 

2. 폴더 동기화

   "도구 메뉴 > 폴더 동기화" 메뉴를 수행하면 폴더 동기화 대화 상자가 띄워집니다. 동기화 하고자하는 폴더를 설정한 후 아래의 스캔 옵션, 비교 옵션을 설정하여 "비교하기" 버튼을 통해 두 폴더 간에 파일을 비교합니다. 비교 후 "동기화" 버튼을 통해 동기화를 수행합니다. 비교 후에는 개별 파일을 선택해 팝업 메뉴로 동기화 옵션을 설정할 수도 있습니다.

   1) 스캔 옵션

파일 포함

비교할 파일 목록에 포함할 파일 이름 (와일드 카드 이용 가능)

파일 배제

비교할 파일 목록에 배제할 파일 이름 (와일드 카드 이용 가능)

동일 파일 제외

동일한 파일은 비교 대상에서 제외합니다.

비동일 파일 제외

동일하지 않은 파일은 비교 대상에서 제외합니다.

하위 폴더 포함

하위 폴더 포함 여부 입니다.

시스템 파일 포함

시스템 파일 포함 여부 입니다.

숨긴 파일 포함

숨긴 파일 포함 여부 입니다.

   2) 스캔 옵션

날짜/시간 비교

파일 날짜/시간에 의해 비교합니다.

속성 비교

파일 속성으로 비교합니다.

크기 비교

파일 크기로 비교합니다.

내용 비교

파일 내용을 비교합니다.
