■ 검색하기

  검색하기는 특정 폴더 내의 특정 문자열로 폴더 및 파일을 찾는 기능입니다. 찾는 위치를 미리 사용자 정의하여 찾고자 하는 폴더를 보다 쉽게 지정할 수 있습니다.

 

1. 검색하기

  검색창은 메인 메뉴 "보기"중 "창 모음"의 "검색창"을 선택하여 띄울 수 있습니다. 또한 단축키 Ctrl+E나 F3을 이용해 검색창을 띄울 수 있습니다. "검색할 파일 및 폴더"에 검색할 문자열을 입력하고, "찾는 위치"를 결정합니다. 그리고 고급 옵션을 선택한 후, "검색시작" 버튼을 누르면 검색이 시작됩니다. 검색하는 도중에 중지하고 싶으면 "검색 중지" 버튼을 눌러서 검색을 중단시킵니다. 간단한 사용자 인터페이스를 지원해 검색하는 중인지 돋보기 그림을 통해 알 수 있습니다. 또한, 검색이 모두 끝나면 검색 결과와 검색 시간을 알 수 있습니다.

 

2. 찾는 위치 설정

   찾는 위치는 다음과 같이 3가지 형태로 설정하여 검색할 수 있습니다. 시스템 폴더, 사용자 정의, 임시적인 폴더로 찾는 위치를 설정할 수 있습니다. 시스템 폴더는 바탕 화면, 내 문서, 드라이브, 로컬 드라이브 등 윈도우 시스템 주요 폴더가 미리 지정되어 있습니다. 그리고 사용자 정의는 여러 드라이브, 경로에 걸쳐 위치된 검색 위치를 한 번에 설정하여 검색할 수 있습니다. 또한 설정 경로는 하위 폴더를 포함할 것인지 여부와 찾을 위치에 포함을 시킬 것인지 배제시킬 것인지도 설정할 수 있습니다. 마지막으로 임시적인 폴더는 임시적으로 폴더를 설정하여 검색합니다.

 

3. 포함하는 텍스트 검색

   포함흔 텍스트 검색은 임의의 파일에 포함되어 있는 텍스트를 검색할 경우 입력하여 검색합니다. 포함하는 텍스트 검색을 할 경우에는 많은 시간이 소모될 수도 있습니다. 그리고 기존 윈도우즈 탐색기에 있는 포함하는 텍스트 검색은 Ansi 텍스트 파일만 검색되었습니다. 하지만 flyExplorer는 Ansi / Unicode 텍스트 파일 모두 검색이 가능합니다.

 

4. 고급 옵션

  고급 옵션은 3가지가 있는데, "하위 폴더 검색"은 찾는 위치를 결정한 폴더의 하위 폴더도 검색할 여부이고, "대/소문자 구분"은 검색할 문자열과 대소문자를 구분하여 검색할 여부입니다. 그리고 "해당 파일 및 폴더 검색"은 검색할 문자열이 완전히 일치하는 파일 및 폴더를 검색할 여부입니다.

 

5. 검색 결과

  검색 시작과 동시에 작업창이 띄워지면서 검색이 진행됩니다. 검색이 모두 끝나거나 검색이 중단되면 현재까지 검색한 결과를 작업창에 검색 결과가 나타납니다. 검색된 파일 및 폴더는 클리보드로 잘라내기와 복사 그리고 드러그 앤 드롭으로 작업을 할 수 있습니다. 또한 파일 수집함으로 넣어 작업 할 수도 있습니다. 검색된 결과 목록을 지우려면 작업창에서 팝업 메뉴를 띄워 "결과 목록 지우기"를 선택하여 지웁니다.
