■ 스타일 옵션

  사용자가 원하는 파일 관리 스타일에 따라 환경 설정의 스타일 옵션을 달리 줄 수 있습니다. 또한 스타일 옵션을 어떻게 설정하느냐에 따라 프로그램 성능이 달라집니다. 대표적인 파일 관리 스타일을 적용하려면 다음과 같습니다.

 

1. 윈도우즈 탐색기

    1) 환경 설정은 다음과 같이 합니다.

        컨텐츠 - 표준 컨텐츠로 설정

        스타일 - 일반 스타일

    2) 폴더창을 보입니다.

    3) 경로바를 모두 닫고, 주소 표시줄을 보입니다.

 

2. 2xExplorer

    1) 환경 설정은 다음과 같이 합니다.

        컨텐츠 - 컨텐츠 사용 안함

        스타일 - 커맨드 스타일

        스타일 - 폴더창 하나만 띄우기

    2) 폴더창 #1을 보입니다.

    3) 주소 표시줄을 모두 숨기고, 경로바와 활성바를 모두 보입니다.

 

3. Total Commader

    1) 환경 설정은 다음과 같이 합니다.

        컨텐츠 - 컨텐츠 사용 안함

        스타일 - 커맨드 스타일

    2) 폴더창을 모두 닫습니다.

    3) 주소 표시줄을 모두 숨기고, 경로바와 활성바를 모두 보입니다.
