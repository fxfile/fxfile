■ 인쇄하기

  인쇄하기는 탐색창에 보여주는 항목을 화면 그대로 인쇄하는 기능입니다. 머리말에는 경로명을 꼬리말에는 페이지 수가 인쇄됩니다.

 

1. 인쇄하기

  인쇄하고자 하는 경로로 이동한 후, 메인 메뉴 "파일"의 "인쇄"를 선택하여 "인쇄" 대화 상자를 띄워 인쇄할 수 있습니다. 옵션은 프린터 종류, 인쇄 범위, 인쇄 매수를 지원합니다.

 

2. 인쇄 설정하기

  인쇄 설정은 메인 메뉴 "파일"의 "인쇄 설정"을 선택하여 "인쇄 설정" 상자를 띄워 프린터, 용지, 인쇄 방향 등을 설정할 수 있습니다.

 

3. 인쇄 미리 보기

  인쇄하기 전에 미리 볼 수 있는 기능으로써, 메인 메뉴의 "파일"의 "인쇄 미리 보기"를 선택하여 볼 수 있습니다.
