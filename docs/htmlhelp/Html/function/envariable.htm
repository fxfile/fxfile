■ 환경 변수 경로

  환경 변수 경로를 이용해 경로를 상대적으로 설정할 수 있습니다.

 

1. 사용 방법

  각 사용자 시스템마다 다른 시스템 폴더를 가질 수 있습니다. 이를 상대적으로 표현하기 위해서는 환경 변수를 이용해 경로를 표현해야 합니다. 이 환경 변수 이용해 경로를 표현할 수 있는 부분은 "연결 항목 설정"의 경로 부분, "경로 이동" 기능, 환경 설정의 "초기 폴더 설정" 등입니다. 예를 들어, A 컴퓨터의 윈도우 시스템 폴더가 'C:\Windows\System32'이고, B 컴퓨터의 윈도우 시스템 폴더가 'D:\Windows\System32' 일 경우, 연결 항목 폴더 설정을 A 컴퓨터에서는 C:\Windows\System32라고 설정하고 B 컴퓨터는 D:\Windows\System32라고 설정해야 합니다. 여기에서 A 컴퓨터의 연결 항목 설정 파일을 B 컴퓨터로 가져갔을 경우에는 B 컴퓨터에 맞게 D:\Windows\System32라고 다시 설정해야 합니다. 이런 류의 폴더를 환경 변수를 이용해 상대적으로 표현하면 특정 컴퓨터에 맞게 다시 설정할 필요는 없습니다. 즉, '%Windows%\System32' 또는 '%System%' 으로 설정하면 상대적 경로로 동작됩니다. 경로 이동, 환경 설정의 초기 폴더 설정도 마찬가지로 설정할 수 있습니다.

 

     ex)

        'C:\Windows\System32' -> '%Windows%\System32' 또는 '%System%'
