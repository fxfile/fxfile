■ 드라이브 도구

  드라이브 도구는 드라이브 간에 빠르게 이동하는 기능으로 구성되어 있는 도구로써 편리함을 제공합니다.

 

 

1. 사용법

  드라이브 도구는 각 드라이브를 연결하는 버튼으로써, 버튼을 누르면 해당 드라이브로 이동하게 됩니다. "환경 설정" 중 "기능"에서 "드라이브 도구로 이동시 폴더 기억하기"를 설정하면, 드라이브 이동시 각 드라이브에서 마지막에 탐색한 폴더를 기억하여, 다시 그 드라이브 이동했을 경우 기억된 폴더로 이동합니다. 또한 옵션을 설정하지 않아도 Ctrl 키 누른 상태에서 해당 드라이브 버튼을 누르면 기억된 폴더로 이동합니다.

 

2. 옵션 설정

  드라이브 도구에서 마우스 오른쪽 버튼을 눌러 팝업 메뉴를 띄워 "도구 이름 짧게 표시"와 "도구 이름 길게 표시"의 옵션을 설정할 수 있습니다. 이 옵션은 드라이브 버튼의 이름을 짧게 표시할 것인지 길게 표시할 것인지 여부입니다. 예를 들어, 위 그림에서 설정된 옵션은 "도구 이름 길게 표시"이고, 짧게 표시한다면 A C D R 로 이름이 표시하게 됩니다.

 

3. 분할창에 각각 할당

  드라이브 도구에서 마우스 오른쪽 버튼을 눌러 팝업 메뉴를 띄워 "드라이브 관련" 중 "분할창에 각각 할당" 옵션을 설정하면, 분할된 창에 드라이브 도구 각각 할당되어 표시됩니다. 그리고 "경로바의 왼쪽에 위치"와 "경로바의 오른쪽에 위치" 옵션을 경로바와 드라이브 도구가 함께 표시할 경우, 한 줄에 둘 다 표시하게 됩니다. 그래서 경로바 왼쪽에 위치할 것인지, 오른쪽에 위치할 것인지의 여부입니다.
