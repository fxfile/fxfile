■ 연결 항목

  연결 항목은 폴더, 파일, 프로그램을 빠르고, 간편하고, 편리하게 실행할 수 있는 기능입니다. 자주 탐색하는 폴더나 자주 여는 파일 및 프로그램을 연결 항목으로 설정하여 보다 편리하게 파일 관리할 수 있습니다.

 

 

1. 연결 항목 추가

  연결 항목을 추가하는 방법은 다음과 같이 3가지를 제공합니다.

     1) 주 메뉴 "도구"의 "연결 항목 편집"을 선택하여, 위 그림 ② 대화 상자를 띄워 설정합니다.

     2) 탐색 창에서 설정하고자 하는 폴더를 드러그 앤 드롭을 이용하여, 위 그림의 ③에서 메인 도구의 "연결 항목" 버튼으로 드롭하여 설정합니다.

     3) 탐색 창에서 설정하고자 하는 폴더를 위 그림 ④ 연결 항목 도구로 드러그 앤 드롭으로 설정합니다.

  이렇게 3가지 방법을 제공하여, 편리하게 연결 항목을 추가할 수 있습니다. 또한 윈도우즈의 시작 버튼 옆에 있는 빠른 실행 도구의 항목들을 위 그림 ② 대화 상자에서 "빠른 실행" 버튼을 이용해 연결 항목으로 추가할 수 있습니다.

 

2. 연경 항목 대상

    1) 파일 - 어떤 임의의 파일을 열 수 있습니다.

    2) 폴더 - 폴더를 보다 쉽게 빠르게 열 수 있습니다.

    3) 프로그램 - 프로그램을 실행할 수 있습니다.

    4) 제어판 - 제어판의 항목을 실행할 수 있습니다.

 

3. 연결 항목 구성 요소

  연결 항목은 "이름", "폴더", "사용자 정의 아이콘", "매개 변수", "시작 경로", "단축키" 로 구성될 수 있습니다.

     1) "이름" - 연결하고자 하는 경로의 별칭(닉네임). [필수, 공용]

     2) "경로" - 연결하고자 하는 파일이나 폴더의 위치입니다. [필수, 공용]

     3) "사용자 정의 아이콘" - 항목의 아이콘을 사용자가 직접 정의합니다. [선택, 공용]

     4) "매개 변수" - 프로그램을 실행할 때 매겨 변수와 함께 할 경우 설정합니다. [선택, 프로그램 전용]

     5) "시작 경로" - 프로그램을 실행할 때 시작 경로와 함께 할 경우 설정합니다. [선택, 프로그램 전용]

     6) "단축키" - 연결 항목을 단축키로 지정하여 보다 편리하게 이용할 수 있습니다. 단축키는 Ctrl+0~9로 지정할 수 있습니다. [선택, 프로그램 전용]

 

4. 연결 항목 실행하기(폴더 열기, 파일 열기, 프로그램 실행)

 설정된 연결 항목을 통해, 폴더 이동하는 방법은 다음과 같이 4가지를 제공합니다.

    1) 위 그림의 ①에서 해당 항목을 마우스 왼쪽 버튼으로 클릭하여 연결 항목을 실행합니다.

    2) 메인 도구의 버튼 중 위 그림의 ③ 처럼 연결 항목 버튼을 눌러 연결 항목을 실행합니다.

    3) 위 그림 ④ 연결 항목 도구에서 해당 연결 항목 버튼을 눌러 연결 항목을 실행합니다.

    4) 연결 항목 설정시 설정한 단축키를 이용하여 연결 항목을 실행합니다.

 

5. 연결 항목과 드러그 앤 드롭 기능

    1) 폴더 - 임의의 폴더 및 파일을 드래그 앤 드롭 기능으로 연결 항목의 폴더 위로 떨어 뜨리면 파일 이동이 됩니다. 그리고 Ctrl 키와 함께 떨어 뜨리면 파일 복사가 됩니다.

    2) 프로그램 - 임의의 파일을 드래그 앤 드롭 기능으로 연경 항목의 프로그램 위로 떨어 뜨리면 그 파일과 함께 프로그램이 띄워집니다. 즉, 프로그램 연동입니다.

 

6. 연결 항목 수정 및 삭제

  연결 항목을 수정하기 위해서는 위 그림 ② "연결 항목 편집" 대화 상자를 띄워, 해당 항목을 선택한 후 "변경" 버튼을 눌러 수정합니다. 그리고 삭제하기 위해서는 위 그림 ④ "연결 항목 편집" 대화 상자를 띄워, 해당 항목을 선택한 후 "삭제" 버튼을 눌러 삭제합니다.

 

7. 환경 변수로 연결 항목 설정

   연결 항목 편집에서 경로를 환경 변수를 이용해 연결 항목의 경로를 설정할 수 있습니다. 윈도우 시스템 폴더를 연결 항목으로 설정하여 사용할 때는 문제는 없겠지만 이 연결 항목 설정을 다른 컴퓨터에 가져갔을 경우에는 설정한 시스템 폴더로 이동이 불가능할 수 있습니다. 이를 해결하기 위해서는 환경 변수를 사용하여 연결 항목을 설정하면 됩니다. 특히 Portable 기능에 유용하게 쓰일 수 있습니다. 참조 : 환경 변수
