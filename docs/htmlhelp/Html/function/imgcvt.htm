■ 일괄적 이미지 포맷 바꾸기

  이미지 파일들을 일괄적으로 특정 포멧으로 바꾸는 기능입니다. 변환해 주는 이미지 파일 포멧은 총 53가지으로 이미지 포멧 변환 전용 프로그램에 못지 않는 기능을 제공합니다.

 

1. 이미지 포멧 변환 방법

  탐색창에서 바꾸고자 하는 이미지 파일을 선택한 후,  메인 메뉴 "파일"의 "이미지 포멧 바꾸기"를 선택하거나 단축키 Ctrl+F2를 통해 "일괄적 이미지 포멧 바꾸기" 대화 상자를 띄워 변환합니다. 바꾸고자하는 포맷을 선택하고 해당 포멧의 옵션을 설정해주면, 이미지 파일이 모두 변환됩니다.
