■ 그림 보기

  그림 보기는 탐색창에서 이미지 파일을 선택하면 "그림 보기" 창으로 이미지을 보여주는 기능입니다. 이미지 파일 포멧은 "환경 설정"의 "썸네일"에서 알 수 있듯이 총 381 여종을 지원합니다. 즉, 이 세상에 존재하는 모든 이미지 중 거의 다 볼 수 있을 정도로 지원합니다.

 

 

1. 2 가지 형태로 그림 보기

  메인 메뉴 "보기"에서 "그림 보기"를 선택하여, 탐색창에서 그림 파일을 선택만으로도 그림을 볼 수 있습니다. 그림을 보는 형태가 2가지로 있어, 사용자의 편의에 따라 선택하여 볼 수 있습니다. 첫 번째는 위 그림과 같이 "그림 보기" 창을 따로 띄워 볼 수 있고, 두 번째는 윈도우즈 탐색기 처럼 볼 수 있습니다. 기본적으로는 첫 번째 방법을 제공합니다. 두번째 방법으로 그림을 볼려면, 위 그림의 "그림 보기" 창에서 팝업 메뉴를 띄워 "탐색창에 도킹하기"를 선택하여 윈도우즈 탐색기처럼 그림을 볼 수 있습니다. 그리고 팝업 메뉴 중 "자동 숨김"은 탐색창에서 이미지 파일을 선택하였을 경우는 "그림 보기"창으로 이미지를 보여주고, 이미지가 아닌 파일을 선택하였을 경우는 숨겨주는 기능입니다.

 

2. 이미지 크기 조절하기

  이미지 크기 조절하기는 확대, 축소, 100%크기, 비율적 크기 조정 4가지가 있습니다. 마우스 오른쪽 버튼을 눌러 위 그림과 같이 팝업 메뉴를 띄워 이미지 크기 조절을 할 수 있습니다. "확대"는 이미지 파일을 조금씩 확대할 수 있고, "축소"는 이미지 파일을 조금씩 축소할 수 있습니다. "100%크기"는 이미지 파일의 원래의 크기로 보여줍니다. 그리고 "비율적으로 크기 조정"은 이미지의 가로와 세로의 비율이 계산되어 "그림 보기" 창을 비율적으로 조절할 수 있습니다.

 

3. 고정된 크기로 그림 보기

  고정된 크기로 그림 보기는 크기 조정, 창 크기로 고정, 100% 크기로 고정 3가지가 있고, 팝업 메뉴를 띄워 설정할 수 있습니다. "크기 고정"은 현재의 보고 있는 이미지 크기로 고정하고, "창 크기로 고정"은 "그림 보기"창을 조절하였을 경우 창에 꽉찬 이미지로 보여줍니다. 그리고 "100% 크기로 고정"은 이미지의 원래 크기로 고정됩니다. 이 3가지의 설정은 전역적으로 설정됩니다. 즉, 어느 이미지 파일 보기에서 설정할 경우, 다음 이미지 파일을 볼 때에도 적용됩니다.

 

4. 바탕 화면의 배경 그림으로 설정

  이미지 파일을 바탕 화면의 배경 그림으로 가운데, 바둑판식 배열, 늘리기 3가지로 설정 가능합니다. 현재 보고 있는 이미지 파일을 배경 그림으로 설정하고 싶을 경우, 팝업 메뉴을 띄워 "바탕 화면의 배경 설정"에서 원하는 메뉴를 선택하면 됩니다.

 

5. 이미지 파일 포멧 지원

  이미지 파일의 지원하는 포멧이 너무 많으므로 이곳에 적지 못하는 것을 양해를 구하며, "환경 설정"의 "썸네일"에서 확인 할 수 있습니다.
