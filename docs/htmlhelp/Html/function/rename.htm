■ 일괄적 이름 바꾸기

 일괄적 이름 바꾸기는 여러 파일이나 폴더의 이름을 일괄적으로 바꿔 주는 기능입니다. 일괄적으로 바꿔주는 포멧 형태 바꿈, 특정 문자 바꿈, 특정 위치 삽입, 특정 위치 삭제, 대/소문자 바꿈 총 5종류로 사용자의 편리에 따라 선택하여 바꿀 수 있습니다. 또한 1.50 부터는 일괄적 포맷과 히스토리를 지원합니다.

1. 사용법

  탐색창에서 바꾸고자 하는 파일을 선택한 후, 메인 메뉴 "파일"의 "이름 바꾸기"를 선택하거나 단축키 F2를 통해 위와 같은 "일괄적 이름 바꾸기" 대화 상자를 띄워 해당 작업을 하면 됩니다. 폴더의 이름을 일괄적으로 바꾸고자 할 경우에는 탐색창에서 Shift 키와 함께 "일괄적 이름 바꾸기" 대화 상자를 띄우면 됩니다. 1.60 이전 버전에는 바꿀 스타일을 모두 한 곳에 넣었는데 너무 복잡해 보이고, 인터페이스가 조잡해 보였는데, 버전 1.60 부터는 바꿀 스타일을 각 탭으로 구성하였습니다. 또한 옵션들이나 히스토리 기능은 툴바로 구성되었습니다.

 

2. 툴바

   최상위에 있는 툴바의 버튼 순서대로 기능이나 옵션을 소개하겠습니다.

     1) 처음 목록 : 대화 상자를 처음으로 띄울 때의 목록으로 초기화합니다.

    2) 되돌리기 : 히스토리 기능으로써, 이전 변경 모드로 되돌리며, "일괄적 포맷에 기록하기" 옵션이 설정되어 있어야 합니다.

     3) 되살리기 : 히스토리 기능으로써, 되돌린 상태에서 다시 되살리며, "일괄적 포맷에 기록하기" 옵션이 설정되어 있어야 합니다.

     4) 위로 이동 : 파일 목록에서 선택된 항목을 한 단계 위로 이동합니다.

     5) 아래로 이동 : 파일 목록에서 선택된 항목을 한 단계 아래로 이동합니다.

     6) 해당 위치로 이동 : 파일 목록에서 선택된 항목을 해당 위치로 이동합니다.

     7) 결과 내에서 바꾸기 : 이 옵션이 설정될 경우에는 변경된 결과 목록에서 바뀌게 되고, 설정되지 않을 경우에는 초기 목록에서 바뀌게 됩니다.

     8) 확장자 변경 안함 : 대상이 파일일 경우 확장를 변경시키 않습니다.

    9) 일괄적 포맷에 기록하기 : 바뀌는 과정을 모두 일괄적 포맷에 기록하여 되돌리기, 되살리기 기능을 수행할 수 있습니다.

 

2. 형식으로 적용

  특정 포멧을 통해 파일 이름을 일괄적으로 바꿔주는 기능으로써, 포멧은 다음과 같습니다.

포멧

설명

/?

자동 숫자 증가의 자리수 (? = [0~9])

/0?

자동 숫자 증가의 자리수(자리수에 맞추어 빈칸은 '0'으로 채움) (? = [0~9])

/n

원본 이름의 파일명

/e

원본 이름의 확장자

/D

현재 날짜

/m

현재 월

ex) 적용할 형태: text_/05-[/D].txt

          text_00000-[2003년 1월 1일].txt

          text_00001-[2003년 1월 1일].txt

          text_00002-[2003년 1월 1일].txt

 

3. 특정 문자 바꿈

  원본 이름에서 특정 문자나 문자열을 새로운 문자열로 대치하는 기능입니다. 옵션은 "반복"과 "대소 문자 구분없이 찾기"을 설정할 수 있습니다. 다음과 같이 예를 들 수 있습니다.

  ex) 원본 이름: text_text_.txt

         "text" -> "텍스트 문서" : 결과는 "텍스트 문서_text_.txt"입니다. 다시 한번 "바꿈" 버튼을 누른 후, 결과는 "텍스트 문서_텍스트 문서_.txt"입니다.

         "_" -> "" : "바꿈" 버튼 누른 후, 결과는 "texttext_.txt"입니다. 다시 한번 "바꿈" 버튼을 누른 후, 결과는 "texttext.txt"입니다.

 

4. 특정 위치 삽입

  특정 위치에 문자열을 삽입하는 기능입니다. 위치와 삽입한 문자열을 입력한 후, "삽입" 버튼을 누르면 특정 위치에 해당 문자열이 삽입됩니다.

 

5. 특정 위치 삭제

  특정 위치에서 특정 길이만큼 삭제하는 기능입니다. 위치와 길이를 입력한 후, "삭제" 버튼을 누르면 특정 위치에 특정 길이만큼 삭제됩니다.

 

6. 대/소 문자 바꾸기

  모두 대문자나 소문자로 바꾸는 기능과 첫 문자만 대문자로 바꾸는 기능입니다. 전체 이름, 이름만, 확장자만을 선택할 수 있는 옵션도 제공합니다.

 

7. 직접 새 이름 편집하기

  리스트에서 해당 파일 및 폴더를 마우스로 더블 클릭하면 새 이름을 편집할 수 있는 대화 상자가 띄워집니다. 새 이름을 입력한 후, 확인 버튼을 누르면 새 이름이 편집한 이름으로 바뀝니다.

 

8. 불규칙적인 이름 편집하기

   위 그림 1에서 "직접 입력" 버튼을 누르면, 위 그림 2와 같이 직접 입력 대화 상자가 열려 불규칙적인 이름을 편집할 수 있습니다. "수직 스크롤 동기화" 옵션은 이전 파일과 새로운 파일의 스크롤 위치를 같게 하는 기능입니다.

 

9. 히스토리

  "일괄적 포맷에 기록하기" 옵션을 설정하고, 위와 같이 UI로 이름 바꾸기를 수행하면 일괄적 포맷에 기록이 됩니다. 이 기록을 이용해 히스토리 기능을 수행할 수 있습니다. 뒤로 가기나 앞으로 가기가 가능합니다. 뒤로 가기 단축키는 Alt + <- 이고, 앞으로 가기 단축키는 Alt + -> 입니다.

 

10. 일괄적 포맷으로 바꾸기

  위와 같이 UI를 설정하여 이름을 바꿀 수 있고, 다음과 같이 일괄적 포맷을 사용하여 이름을 바꿀 수도 있다.

1) 형식으로 적용: <1:F1:F2:F3>

F1

시작 번호

F2

번호 증가

F3

형식(위 2번 형태로 적용 참고)

2) 특정 문자 바꿈: <1:F1:F2:F3:F4>

F1

반복 횟수

F2

찾을 문자

F3

대소 문자 구분 없이 찾기(0 - 구분하지 않음, 1 - 구분함)

F4

바꿀 문자

3) 특정 문자 삽입: <1:F1:F2:F3>

F1

삽입할 타입(0 - 맨 앞에, 1 - 맨 뒤에, 2 - 특정 위치에)

F2

삽입할 위치

F3

삽입할 문자

4) 특정 위치 삭제: <1:F1:F2:F3:F4>

F1

삭제할 타입(0 - 맨 앞에, 1 - 맨 뒤에, 2 - 특정 위치에)

F2

삭제할 위치

F3

삭제할 길이

5) 대/소문자 바꿈: <1:F1:F2>

F1

대/소문자로 바꿀 부분(0 - 이름만, 1 - 확장자만, 2 - 전체 이름)

F2

대/소문자로 바꿀 타입(0 - 모두 소문자로, 1 - 모두 대문자로, 2 - 첫 글자만 대문자로, 3 - 단어의 첫 글자만 대문자로)

 

예) 일괄적 포맷의 예는 다음과 같다.

     <1:0:1:test_/05 [/D]>

     <2:1:0:윈도우즈 탐색기:flyExplorer>

     <3:0:0:[flyExplorer] ->

     <4:2:0:10>

     <5:2:3>
