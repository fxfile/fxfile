■ 일괄적 새로 만들기

  일괄적 새로 만들기는 폴더나 특정 파일을 연속적인 번호와 포맷 형식으로 일괄적으로 만들어 줍니다.

 

1. 사용법

  메뉴 "파일"에서 "새로 만들기"의 "일괄적 새로 만들기"를 선택하면, "일괄적 새로 만들기" 대화 상자가 띄워집니다.

 

2. 설정하기

옵션

설명

폴더로 새로 만들기

폴더를 새로 만들 것인지 여부입니다. 이 옵션을 설정하지 않으면 파일을 일괄적으로 만듭니다.

이름

새로 만들 폴더나 파일명을 적습니다. 이름 바꾸기의 포맷 형식과 호환됩니다.

숫자 증가값

연속될 숫자의 증가값

시작 번호

연속될 숫자의 시작 번호

끝 번호

숫자의 끝 번호

번호 개수

시작 번호부터의 번호 개수

자리수에 맞게 빈자리 '0'으로 채우기

자리수에 맞게 빈곳에 '0'을 채움

 

  예를 들어 "이름"에 "TEST/05"라고 입력하고, "폴더로 새로 만들기" 옵션을 설정하고, "시작 번호"를 "0", "끝 번호"를 "100"으로 주면, 다음과 같이 새로운 폴더들이 만들어 집니다.

    TEST00000

    TEST00001

    TEST00002

    ...

    ...

    TEST00098

    TEST00099

    TEST00100
