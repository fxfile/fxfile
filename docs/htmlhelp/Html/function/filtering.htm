■ 필터링 기능

  필터링은 확장자에 따라 설정되며, 탐색창의 파일이나 폴더를 그룹으로 묶어 글자색과 아이콘을 적용할 수 있습니다. 추후에는 탐색창에서 특정 확장자만 보이는 필터링 기능을 더 추가할 예정입니다.

 

1. 구성 요소

    1) 이름 - 구성할 필터링의 별칭(닉네임)입니다.

    2) 확장자 - 파일의 확장자입니다. 여러 확장자로 구성하려면 세미콜론(;)으로 구분해 줍니다. 예를 들어 "txt;hwp;doc" 이런식으로 적습니다.

    3) 색 - 확장자로 구성된 필터링의 표시색입니다.

    4) 아이콘 - 확장자로 구성된 필터링의 사용자 정의 아이콘입니다.

 

2. 필터링 설정하기

   "환경 설정"의 "탐색창" 중 "필터링"에서 확장자에 따른 필터링을 설정할 수 있습니다. "추가", "편집", "삭제", "위로", "아래로" 버튼을 이용해서 위 구성 요소에 맞게 설정하면 됩니다. 단, 주의할 점은 예약된 필터링으로써, 첫 번째 필터링은 폴더이고, 두 번째 필터링은 일반(기타) 파일이어야 하고, 절대 지워서는 안됩니다. 폴더와 일반 파일은 프로그램 내부적으로 고정된 위치를 참고하기 때문에 색은 수정해도 되나, 확장자 수정, 삭제, 위치 변경을 해서는 절대로 안됩니다.

 

3. 설정한 필터링 사용하기

    1) 필터링에 따른 색 사용하기 - "환경 설정"의 "탐색창" 중 "필터링"에서 "필터링에 따른 색 사용" 옵션을 설정하면, 탐색창에서 같은 종류의 파일을 화려한 색으로 표시됩니다.

    2) 사용자 정의 아이콘 사용하기 - "환경 설정"의 "탐색창" 중 "표시"에서 "사용자 정의 아이콘 사용하기" 옵션을 설정하면, 탐색창에서 사용자가 정의된 아이콘을 사용할 수 있습니다.

 

4. 사용자 정의 아이콘

   사용자 정의 아이콘은 기본적으로 제공해 드립니다. flyExplorer가 설치된 경로에 보면 "custom_image_list_16.bmp"와 "custom_image_list_32.bmp" 라는 파일이 있습니다. 이 파일은 17 개 아이콘으로 구성된 16비트 컬러 비트맵 파일입니다. 사용자가 직접 아이콘을 디자인해서 넣고 싶다면 다음과 같이 주의해야 할 점이 있습니다.

   1) 비트맵 파일이어야 하고, 컬러수는 16비트 컬러이어야 합니다.
