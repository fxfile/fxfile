■ 창 분할

  창 분할은 두 개로 분할 시켜 폴더간의 작업을 더욱 편리하게 할 수 있는 기능입니다.

 

1. 창 분할

  창 분할은 3가지 형태로 분할할 수 있다. 단일창, 수직 분할 창, 수평 분할 창으로 폴더창과 탐색창을 각각 2개씩 분할 시킬 수 있습니다. 다음 도식화된 것처럼 볼 수 있습니다.

  참고적으로 폴더창은 도킹을 지원하기 때문에 어느 곳에서나 위치할 수 있습니다. 탐색창 왼쪽이나 오르쪽, 위, 아래에 위치할 수도 있고, 폴더창을 따로 띄울 수도 있습니다.

 

2. 옵션

   1) 같은 비율로 창 분할 : 탐색창의 분할을 같은 비율의 크기로 조정합니다.

   2) 수평 분할시 크기 동기화 : 위 도식화된 수평 분할 창 상태에서 탐색창이나 폴더창의 크기를 조정할 경우 탐색창과 폴더창의 크기를 같게 유지시켜 줍니다.

   3) 스왑하기 : 창 #1과 창 #2에서 각각 탐색하고 있는 폴더를 서로 바꿔 줍니다.

 

3. 창간의 파일 작업

   1) 다른 창으로 복사 : 현재 창에서 선택한 파일이나 폴더를 다른 창으로 복사합니다.

   2) 다른 창으로 이동 : 현재 창에서 선택한 파일이나 폴더를 다른 창으로 이동합니다.

   3) 창 #1로 복사 : 창 #2에서 선택한 파일이나 폴더를 창 #1로 복사합니다.

   4) 창 #1로 이동 : 창 #2에서 선택한 파일이나 폴더를 창 #1로 이동합니다.

   5) 창 #2로 복사 : 창 #1에서 선택한 파일이나 폴더를 창 #2로 복사합니다.

   6) 창 #2로 이동 : 창 #1에서 선택한 파일이나 폴더를 창 #2로 이동합니다.
