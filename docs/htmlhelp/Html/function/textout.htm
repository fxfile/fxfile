■ 파일 목록 만들기

  탐색창에서 선택한 파일 및 폴더의 이름이나 경로명를 지정한 텍스트 문서로 내보내는 기능으로써, 파일 목록을 쉽게 만들 수 있습니다.

 

1. 사용 방법

  탐색창에서 텍스트 문서로 출력하고자 하는 파일 및 폴더를 선택한 후, 메인 메뉴 "파일"중 "내보내기"의 "선택한 항목을 텍스트 문서"를 선택하면, "텍스트 파일로 내보내기" 대화 상자가 띄워집니다. 또한 단축키 Ctrl+T를 눌러서 띄울 수도 있습니다. "텍스트 파일로 내보내기"를 띄우기 전에 저장할 경로명을 지정하는 상자가 먼저 띄워집니다. 경로명을 지정하고 필요한 옵션을 선택한 후 "확인" 버튼을 누르면 텍스트 문서로 파일 목록이 만들어집니다.

 

2. 옵션 설정

  편의에 따라 옵션을 지정하여 파일 목록 출력할 수 있습니다. 지원하는 옵션은 다음과 같습니다.

옵션

설명

파일만 출력

폴더를 제외한 파일만 출력

확장자 포함

파일 중 확장자가 포함되어 있을 경우 확장자를 제외한 파일명만 출력

경로명 붙이기

파일 및 폴더의 전체 경로와 함께 출력

라인별 출력

항목 간에 라인 별로 출력

크기, 종류, 날짜 포함

이름, 크기, 종류, 수정된 날짜까지 포함하여 출력

구분자로 출력

이름, 크기, 종류, 수정된 날짜를 포함여 출력시 구분은 구분자로 하여 출력
