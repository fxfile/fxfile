■ 작업 폴더

  작업 폴더 기능은 빈번하게 탐색하는 폴더를 등록하여 빠르게 이동하는 기능입니다.

 

1. 작업 폴더 설정

    1) 해당 폴더로 이동하여 Ctrl키+Shift키와 함께 작업 폴더 1, 2, 3, 4, 5 메뉴를 클릭하여 설정합니다.

    2) 작업 폴더 1, 2, 3, 4, 5 툴바 버튼에 드래그 앤 드롭을 이용해 설정합니다.

 

2. 작업 폴더로 이동하기

   작업 폴더 1, 2, 3, 4, 5 메뉴를 클릭하여 이동합니다.

 

3. 작업 폴더 모두 초기화하기

   작업 폴더 모두 초기화 메뉴를 수행하면 모두 초기화 됩니다.
