■ 환경 설정 > 창 스타일

  환경 설정 중 썸네일은 탐색창의 보기 스타일을 썸네일 이미지로 할 경우, 썸네일에 관련된 설정을 할 수 있게 구성되어져 있습니다. 스타일 옵션에 대한 자세한 설명은 특징적인 기능 둘러보기에서 참고하세요.

설명

일반 스타일

일반적으로 윈도우즈 탐색기 스타일로 설정합니다.

고급 스타일

커맨드 스타일 등 고급 스타일로 설정합니다.

  + 커맨드 스타일

폴더창에 의존하지 않고 탐색창에서 폴더를 탐색함로써 토탈 커맨드와 같은 스타일로 설정하여 일반 스타일보다 더욱 빠르게 탐색할 수 있습니다.

  + 폴더창 하나만 띄우기

분할 창에서 폴더창을 하나만 띄워 대표적으로 2xExplorer와 같은 스타일로 설정합니다. 또한 탐색창이 폴더창에 의존하지 않기 때문에 일반 스타일에 비해 더욱 빠르게 탐색할 수 있습니다.

    + 폴더창과 탐색창 간에 연동

"폴더창 하나만 띄우기" 옵션이 활성된 상태에서 폴더창과 탐색창 간에 연동할 수 있습니다.
