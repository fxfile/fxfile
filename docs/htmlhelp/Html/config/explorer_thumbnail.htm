■ 환경 설정 > 탐색창 > 썸네일

  환경 설정 중 썸네일은 탐색창의 보기 스타일을 썸네일 이미지로 할 경우, 썸네일에 관련된 설정을 할 수 있게 구성되어져 있습니다.

설정

설명

너비

썸네일 이미지의 너비를 설정합니다.

높이

썸네일 이미지의 높이를 설정합니다.

우선 순위

썸네일 이미지 읽기의 우선 순위를 설정합니다. 환경 설정에 주의와 같이 우선 순위가 높을 수록 사용자 인터페이스가 느릴 수 있습니다.

지원하는 이미지

지원하는 이미지의 포멧을 보여줍니다.

확장자로만 해석

썸네일이나 그림보기에서 이미지를 로드할 때 확장로만 해석할 것인지 여부입니다. 이 옵션을 설정하면 썸네일이나 그림 보기에서 성능이 향상됩니다.

캐쉬 저장

썸네일 이미지 캐쉬에 있는 내용을 종료시 저장합니다.

캐쉬 정리

썸네일 이미지 캐쉬에 있는 내용 중 삭제된 파일의 이미지를 정리합니다.

캐쉬 지우기

썸네일 이미지 캐쉬에 있는 내용을 모두 지웁니다.
