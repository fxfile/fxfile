■ 환경 설정 > 탐색창 > 표시

  환경 설정의 탐색창 중 표시는 탐색창에서 표시할 다양한 옵션을 설정할 수 있게 구성되어져 있습니다.

설정

설명

상위 폴더 [..] 표시하기

탐색창에 상위 폴더 이동할 [..]을 표시할 것인지 여부입니다.

속성 ARHS 스타일로 표시

컨텐츠에서 특성이 아닌 속성으로 ARHS 스타일로 표시할 것인지 여부입니다.

드라이브 표시

탐색창에서 사용자의 모든 드라이브를 표시할 것인지 여부입니다.

  + 드라이브 용량 표시

"드라이브 표시" 옵션이 설정된 상태에서, 각 드라이브의 용량을 표시합니다.

드라이브 항목

탐색창에서 드라이브 항목을 표시할 것인지 여부입니다.

24시간제 시간 표시

시간을 24시간제로 표시할 것인지 여부입니다.

2자리 연도 사용

날짜의 연도를 2자리 연도를 표시할 것인지 여부입니다.

그리드 라인

탐색창을 그리드 라인의 설정으로 항목 간의 라인으로 구분을 해 줄 것인지 여부입니다.

전체 행 선택

탐색창에서 항목 선택시 해당 항목의 행을 전체 선택할 것인지 여부입니다.

사용자 정의 아이콘 사용하기

필터링에 따른 아이콘의 설정을 탐색창에서 사용할 것이진 여부입니다.

항목 표시

항목을 표시할 때 폴더과 파일을 표시할 것인지, 폴더만, 파일만 표시할 것인지 설정합니다.

대/소문자

항목(이름, 크기, 종류, 날짜, 속성)을 대/소문자로 표시할 것인지 설정합니다.

용량 표시 단위

용량 표시 단위를 설정합니다.
