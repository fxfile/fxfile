설정

설명

다중 선택하여 이름 변경시 일괄적 이름 바꾸기 기능으로 동작시킵니다.

   + 폴더 포함하기

다중 선택하여 일괄적 이름 바꿀시 폴더도 포함시킵니다.

마우스로 이름 바꾸기

마우스로 이름 바꾸기를 허용합니다.

 

2. 단일 이름 바꾸기 방식

설정

설명

기본 방식

단일 이름 변경시, 윈도우즈 탐색기와 같은 기본 방식으로 이름을 변경합니다.

대화 상자 방식

단일 이름 변경시, 대화 상자를 띄워 이름을 변경합니다.

일괄적 이름 바꾸기 방식

단일 이름 변경시, 일괄적 이름 바꾸기 기능으로 이름을 변경합니다.

 

3. 단일 확장자 변경 방식

설정

설명

기본 방식

단일 파일 이름 변경시, 확장자에 대해 고려하지 않습니다.

확장자 기억 방식

단일 파일 이름 변경시, 확장자를 표시하지 않고 이름 변경 완료시 기억된 확장자를 덮붙입니다.

파일 이름 선택 방식

단일 파일 이름 변경시, 윈도우즈 비스타 탐색기와 유사하게 파일 이름만 선택시킵니다.
