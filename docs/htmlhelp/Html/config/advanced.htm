■ 환경 설정 > 고급

  환경 설정 중 고급적인 설정을 할 수 있도록 구성되어져 있습니다. 각종 고급 설정과 시스템 트레이에 표시로 구성되어져 있습니다.

설정

설명

고급 설정

고급적인 환경 설정입니다.

  + flyExplorer 하나만 띄우기

flyExplorer를 하나의 프로그램만 띄울지 여부입니다.

  + 종료시 확인후 종료하기

실수로 방지로, 프로그램 종료시 확인을 받고 종료할 것인지 여부입니다.

  + 마우스의 이동 버튼이 동작되지 않을 경우 활성화

마우스의 이동 버튼이 동작하지 않을 경우 활성화할 것인지 여부입니다.

  + 최근 실행된 파일 목록 사용하기

최근 실행된 파일 목록을 사용할 것인지 여부입니다.

  + 윈도우 탐색기에 "flyExplorer로 열기" 등록

윈도우 탐색기에서 특정 폴더의 팝업 메뉴를 띄워 "flyExplorer로 열기" 기능을 수행하여, 그 폴더로 flyExplorer로 열 수 있는 기능을 등록시킵니다.

시스템 트레이

시스템 트레이이 관련된 설정입니다.

  + 시스템 트레이에 표시하기 안함

시스템 트레이에 표시하지 않습니다.

  + 시스템 트레이에 표시

시스템 트레이에 표시합니다.

    + 닫기 했을 때, 종료하기 않고 숨기기

프로그램 종료 명령을 했을 경우, 종료되지 않고 시스템 트레이에 숨겨 줍니다.

최소화시, 트레이로 표시

프로그램 최소화시, 시스템 트레이로 표시합니다.

복원시, 초기 폴더로 이동

시스템 트레이로부터 복원시 환경 설정에서 설정한 초기 폴더로 이동합니다.

한번 클릭으로 복원/숨기기

시스템 트레에서 한번 클릭으로 복원시킵니다.

최근 실행된 목록 지우기

파일 메뉴에 표시되는 최근 실행된 파일 목록을 모두 삭제합니다.

기본값 복원

고급 설정과 시스템 트레이의 설정을 모두 기본값을 돌려 놓습니다.
