■ 환경 설정 > 기능 > 드래그 드롭

 탐색창이나 폴더창에서 사용되는 드래그 앤 드롭 기능에 옵션을 설정할 수 있게 구성되어져 있습니다.

1. 드래그 시작 옵션

설정

설명

사용 안함

드래그 기능을 사용 금지합니다.

기본 드래그 시작

윈도우즈 탐색기처럼 기본적인 드래그 시작합니다.

Ctrl키 누르면서 드래그 시작

실수 방지를 위해 드래그를 시작할 경우 항상 Ctrl 키를 눌러 시작합니다.

이동 거리에 따른 드래그 시작

실수 방지를 위해 대상을 마우스 왼쪽 버튼을 누르고 이동 거리에 따라 드래그를 시작합니다.

 

2. 드래그 중

설정

설명

폴더 펼치는 시간

폴더창에서 드래그하는 도중 폴더를 펼치는 시간을 설정합니다.

스크롤 시간

폴더창이나 탐색창에서 스크롤 시간을 설정합니다.

설명
