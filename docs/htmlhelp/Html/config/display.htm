■ 환경 설정 > 표시

  환경 설정 중 표시는 사용자 정의 글꼴 등의 UI에 대한 설정할 수 있게 구성되어져 있습니다.

설정

설명

사용자 정의 글꼴 사용

폴더창, 탐색창, 경로바, 주소 표시줄에서 시스템 기본 글꼴이 아닌 사용자 정의 글꼴을 사용할 것인지 여부입니다.

메뉴 에니메이션 효과 사용하기

메뉴를 띄울 때 에니메이션 효과를 사용할 것인지 여부입니다.

표준 메뉴 스타일 사용하기

MS 오피스 2003 메뉴 스타일을 사용할 것인지, 표준 메뉴 스타일을 사용할 것인지 여부입니다.

활성화된 창 표시색

활성화 된 창의 표시색 즉, 활성바의 색을 설정합니다. 분할된 창에서 포커스 상태를 나타낼 때 유용합니다.

경로바에서 아이콘 표시 경로방에서 폴더 아이콘을 표시할 것인지 여부입니다.

경로바에서 실제 경로로 표시하기

경로바에서 폴더를 실제 경로로 표시할 것인지 여부입니다.

경로바 강조색 사용

경로바에서 마우스를 올려 놓았을시 강조될 색을 설정합니다.
