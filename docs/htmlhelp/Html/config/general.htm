■ 환경 설정 > 일반

  환경 설정 중 일반적인 설정을 할 수 있습니다. 예를 들어, 각 폴더 보기 기억, 숨김 파일 및 폴더 표시, 확장자 표시 등의 기본적인 설정들로 구성되어져 있습니다.

한 번 클릭해서 열기

탐색창에서 한 번 클릭해서 항목을 열거나 실행할 것인지 여부입니다.

두 번 클릭해서 열기

탐색창에서 두 번 클릭해서 항목을 열거나 실행할 것인지 여부입니다.

숨김 파일 및 폴더 표시

숨김 파일 및 폴더를 표시할 것인지 여부입니다.

보호된 운영체제 파일 표시 보호된 운영체제 파일 및 폴더를 표시할 것인지 여부입니다.

제목 표시줄에 전체 경로 표시

제목 표시줄에 현재 탐색하고 있는 폴더의 경로를 전체 경로명으로 표시할 것인지 여부입니다.

주소 표시줄에 전체 경로 표시

주소 표시줄에 현재 탐색하고 있는 폴더의 경로를 전체 경로명으로 표시할 것인지 여부입니다.

파일 및 폴더 팝업 설명 표시

탐색창에서 파일 및 폴더의 설명을 팝업으로 표시할 것인지 여부입니다.

  + 파일명 및 폴더명 표시

팝업 설명에 파일명 및 폴더명을 표시합니다.

새 항목을 대화 상자로 만들기

새 폴더, 새 텍스트 파일 등 새 항목을 만들시 대화 상자를 띄워 만듭니다.

확장자 표시

파일의 확장자 표시 옵션을 설정합니다.
