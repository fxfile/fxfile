■ 환경 설정 > 기능

 환경 설정 중 기능은 프로그램의 전체적인 기능에 대한 옵션을 설정할 수 있게 구성되어져 있습니다.

설정

설명

드라이브별 마지막 폴더 기억하기

드라이브별 마지막으로 탐색했던 폴더를 기억하여 다시 그 드라이브로 이동시 기억된 폴더로 이동합니다.

Shift 키와 함께 드라이브 이동하기

드라이브 이동시 Shift 키와 함께 특정 알파벳 키를 누르면 해당 드라이브로 이동합니다.

주소 표시줄에서 인터넷 연결 호환

주소 표시줄에서 인터넷 경로를 입력시 그 경로와 함께 웹 브라우저를 띄워 인터넷으로 연결시킵니다.

작업창 닫을시 검색 결과 목록 지우기

작업창을 닫을시 검색 결과 목록을 지울 것인지 여부입니다.

윈도우 탐색기와 같은 새로 만들기 항목 표시 윈도우 탐색기에서 표시되는 새로 만들기 항목을 모두 표시합니다.
파일 목록 만들시, 현재 폴더를 기본 경로로 설정하기 파일 목록 만들시, 마지막 폴더를 기억하는 것이 아니라 항상 현재 폴더를 기본 경로로 설정합니다.
작업 폴더 가능하면 실제 경로로 설정하기 작업 폴더 설정시 가능하면 실제 경로로 설정합니다.

이름 복사시 구분자 설정

파일이나 폴더의 이름을 클립보드로 복사할 경우 이름과 이름 사이에 구분자를 설정합니다. 한 줄에 하나의 이름으로 복사할 경우 "\r\n"을 설정하면 됩니다. 탭으로 구분할 경우에는 "\t" 로 설정하면 됩니다.
