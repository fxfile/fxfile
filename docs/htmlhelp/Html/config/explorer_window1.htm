■ 환경 설정 > 탐색창 > 창 #1

  환경 설정의 탐색창 중 창 #1은 분할된 창 #1 전용 옵션을 설정할 수 있게 구성되어져 있습니다.

설정

설명

사용자 정의 탐색창

사용자 정의(배경 그림, 글자색, 배경색) 탐색창을 꾸밀 것인지 여부입니다.

  + 배경 그림

탐색창에 대한 배경 그림을 설정합니다.

  + 글자색

탐색창에 대한 글자색을 설정합니다.

  + 배경색

탐색창에 대한 배경색을 설정합니다.

초기 폴더 설정 안함

탐색창의 초기 폴더를 설정하지 않을 것인지 여부입니다.

마지막 폴더 기억하기

마지막으로 탐색창에서 작업했던 폴더를 초기 폴더로 지정할 것인지 여부입니다.

  + 네트워크 폴더 기억하지 않기

마지막 폴더가 네트워크 폴더라면 기억하지 않을 것인지 여부입니다.

시작 폴더

탐색창의 시작 폴더를 설정할 것인지 여부와 시작 폴더를 설정합니다.
