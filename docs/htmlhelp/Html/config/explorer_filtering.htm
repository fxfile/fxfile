■ 환경 설정 > 탐색창 > 필터링

  환경 설정의 탐색창 중 필터링은 확장자에 따른 필터링을 설정하고 그 필터링에 따른 아이콘, 표시색도 설정할 수 있게 구성되어져 있습니다. 필터링에 대한 자세한 설명은 특징적인 기능 둘러보기를 참고하세요.

설정

설명

필터링에 따른 색 사용

필터링 설정에 따른 색을 탐색창에서 사용할 것인지의 여부입니다.

  + 글자색

필터링 설정에 따른 색을 파일명이나 폴더명의 글자색으로 사용합니다.

  + 배경색

필터링 설정에 따른 색을 파일명이나 폴더명의 배경색으로 사용합니다.

필터링 리스트

필터링의 목록을 나열해줍니다. 1번째 폴더 필터링과 2번 일반 파일 필터링은 고정되어져 있으므로, 삭제할 수 없고, 일부 수정만 가능합니다.

위로

필터링 리스트에 선택된 필터링의 순서를 위로 이동시킵니다.

아래로

필터링 리스트에 선택된 필터링의 순서를 아래로 이동시킵니다.
기본값 복원 필터링 리스트를 초기화 합니다.

추가

설정할 필터링을 추가합니다.

편집

필터링 리스트에 선택된 필터링에 대해 이름과, 확장자, 색, 아이콘 등을 편집합니다. 필터링 설정 대화 상자에서 직접 입력에 255,255,255 와 #FFFFFF 형식의 컬러를 직접 입력하고 설정 버튼을 누르면 왼쪽 색에 적용됩니다.

삭제

필터링 리스트에 선택된 필터링을 삭제합니다.
