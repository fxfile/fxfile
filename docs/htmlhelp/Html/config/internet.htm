■ 환경 설정 > 인터넷

  환경 설정 중 인터넷 탭 브라우저에 관련된 설정을 할 수 있게 구성되어져 있습니다.

설정

설명

탭 하단에 위치

탭의 위치를 하단에 위치할 것인지 여부입니다.

탭 크기

인터넷 탭 브라우저의 탭 크기를 설정합니다.

모든 팝업창 띄우지 않기

웹페이지 로딩시 팝업창을 모두 띄우지 않습니다.

모든 팝업창 내부로 띄우기

모든 팝업창을 내부로 띄울 것이지 여부입니다. 설정하지 않으면 인터넷 익스플로러와 같은 형태로 띄워집니다.

모든 웹 문서를 외부로 띄우기

모든 웹페이지를 외부로 띄울지, 아니면 flyExplorer의 웹 탭 브라우저로 띄울지 여부입니다.

내부로 팝업창 띄울시 활성화

내부로 팝업창을 띄울시 띄운 창을 활성화시킬 것인지 여부입니다.

닫기시 다음 페이지 활성화

현재 창을 닫을시 이전 페이지를 활성화시킬 것인지, 다음 페이지를 활성화시킬 것인지 여부입니다.

즐겨찾기 새 페이지에 로드하기

즐겨찾기의 웹 사이트를 로드할 때 새 페이지에 로드할 것인지 여부입니다.

마우스 왼쪽 버튼을 더블 클릭으로 닫기

마우스 왼쪽 버튼을 더블 클릭으로 현재창을 닫을 것인지 여부입니다.

전환시 즐겨찾기창 띄우기

인터넷 탭 브라우저로 전환시 즐겨찾기창을 자동으로 띄울지 여부입니다.

전환시 웹 저장

웹 탭 브라우저에서 파일 관리로 전환시 웹 상태를 모두 초기화시키지 않고 저장하여 다시 전환시 저장된 웹 상태를 보여줍니다.

주소 표시줄에 사이트 주소 입력시 웹 전환

파일 관리에서 주소 표시줄에 인터넷 웹 주소를 입력할 경우, 인터넷 탭 브라우저로 전환할 것인지 여부입니다.
