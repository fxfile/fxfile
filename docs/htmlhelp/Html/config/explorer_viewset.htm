■ 환경 설정 > 탐색창 > 보기 설정

  환경 설정의 탐색창 중 보기 설정은 탐색창의 보기 스타일, 컬럼, 정렬 등을 설정하는 보기 설정을 설정할 수 있게 구성되어져 있습니다.

설정

설명

폴더 보기 설정

폴더 보기 설정의 스타일을 설정합니다.

  + 보기 설정 기억하지 않기

폴더의 보기 설정을 기억하지 않습니다.
    + 보기 스타일 저장 폴더의 보기 스타일을 저장합니다.

  + 기본 보기 설정 기억하기

폴더의 기본 보기 설정으로 기억합니다. 분할된 탐색창간 각각 보기 설정을 기억합니다.

  + 모든 창 동일하게 보기 설정 기억하기

폴더의 보기 설정을 분할된 탐색창을 구분하지 않고 모두 동일하게 기억합니다.

  + 폴더마다 보기 설정 기억하기

각 폴더별로 보기 설정을 기억합니다.

초기 보기 스타일

폴더마다 보기 설정 기억하기 옵션 설정시, 처음으로 탐색하는 폴더의 초기 보기 스타일을 설정합니다.

초기 정렬

폴더마다 보기 설정 기억하기 옵션 설정시, 처음으로 탐색하는 폴더의 초기 정렬을 설정합니다.

초기 정렬 순서

폴더마다 보기 설정 기억하기 옵션 설정시, 처음으로 탐색하는 폴더의 초기 정렬 순서를 설정합니다.

정렬 안함

보기 설정 중 정렬을 하지 않습니다.

컬럼폭 자동 조절

컬럼 중 이름 컬럼에 대해서만 컬럼 폭을 자동으로 정렬합니다.

종료시 보기 설정 정리하기

프로그램 종료시, 기억된 보기 설정에 대해 유효하지 않는 보기 설정을 정리합니다.

보기 설정 정리하기

기억된 보기 설정에 대해 유효하지 않는 보기 설정을 정리합니다.

보기 설정 초기화

기억된 보기 설정을 모두 초기화합니다.
